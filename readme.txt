== Description ==

A WooCommerce plugin that allow user to customize the Add to cart button and add a simple "Buy Now" button.

In WooCommerce side menu, there will be a new submenu link called 'TT Add to Cart Buy Now' where you can:

*   Enabled / Disabled both "Add to Cart" and "Buy Now" options
*   Customize "Add to cart" button text and page URL for all simple and variable products with a custom cart icon
*   Add a "Buy Now" button with a custom arrow icon
*   Customize "Add to cart" and Buy Now button at the shop page
*   Skip Cart and Redirect to Checkout option for Add to Cart button


= Features =

*   Customize "Add to cart" button at both Shop and Product page
*   Add a "Buy Now" button at both Shop and Product page
*   2 languages available : English US (en_US) and Chinese (zh_CN)

== Installation ==

1. Upload the entire *woocommerce-add-to-cart-buy-now* folder to the */wp-content/plugins/* directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'WooCommerce' at left sidebar menu and update the options at Tab *TT Add to Cart Buy Now* there.
4. That's it. You're ready to go and cheers!
